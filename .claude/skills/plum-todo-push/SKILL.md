# Todo Push

Append a new task to the end of TODO.txt.

## Instructions

1. The user's arguments after `/todo-push` are the task text to add
2. Read TODO.txt to get the current contents
3. Append the task text as a new line at the end of TODO.txt
4. Confirm to the user what was added and show the updated TODO.txt
